想让常用电脑自动上线？

下面的一键脚本适合经常需要被访问的电脑。复制命令执行后，它会一步步引导你设置地址和固定会话名，然后让这台电脑在登录后自动启动 DeskGo。

macOS / Linux 一键安装

适合个人 Mac、Linux 台式机或笔记本。安装后，这台电脑登录桌面时会自动启动 DeskGo。

  • 支持引导式安装 / 卸载，也支持一次性传参
  • 可自定义 DeskGo 地址、固定会话名与画面设置
  • 适合希望“开机登录后就能连上”的常用电脑
GitHub shell 脚本 查看完整说明
直接复制这条安装命令 
rm -f /tmp/deskgo-autostart.sh
curl -fsSL -H 'Cache-Control: no-cache' "{{ .AutostartShellURL }}?ts=$(date +%s)" -o /tmp/deskgo-autostart.sh
bash /tmp/deskgo-autostart.sh install

Windows 一键安装

适合办公电脑或家中常用 Windows 设备。安装后，登录桌面时会自动启动 DeskGo，无需管理员权限。

  • 支持引导式或一次性传参安装，也支持卸载清理
  • 自动下载适合当前 Windows 架构的客户端
  • Windows 自动运行时，跟着安装器的默认推荐设置即可
GitHub PowerShell 脚本 查看完整说明
直接复制这条安装命令 
$script = Join-Path $env:TEMP 'deskgo-autostart.ps1'
$uri = "{{ .AutostartPowerShellURL }}?ts=$([DateTimeOffset]::UtcNow.ToUnixTimeSeconds())"
Remove-Item $script -ErrorAction SilentlyContinue
Invoke-WebRequest $uri -UseBasicParsing -Headers @{ 'Cache-Control' = 'no-cache' } -OutFile $script
powershell -ExecutionPolicy Bypass -File $script install -Codec h264

安装前先准备好这两件事

这样安装过程会更顺畅，也更方便你以后反复使用。

  • 一个你以后容易记住的会话名，例如 home-pc 或 office-main
  • 能正常访问的 DeskGo 地址，以及一台会保持桌面登录状态的电脑
  • 如果你只是偶尔使用，先手动启动客户端往往更简单

这些日常场景最适合 DeskGo。

如果你脑子里想的是“我想在别处打开那台电脑”，通常就是下面这几种情况。

远程打开家里的电脑

出门在外时，随时回到家里那台电脑，拿文件、开程序、做一点临时操作。

  • 适合出差、旅行、外出办公时使用
  • 浏览器就能进入，不用再装查看端
  • 固定会话名后更容易反复连接

继续办公室里的工作

下班后或外出时，从浏览器回到自己的工位机或常用工作电脑，继续处理任务。

  • 适合固定工位机和个人工作站
  • 自动运行后，登录桌面就能上线
  • 同一个会话名可以长期使用

帮家人或同事看电脑

当对方不熟悉技术细节时，你只需要让对方先把电脑连上，再通过浏览器进入查看。

  • 可以直接分享完整访问链接
  • 等待和断线状态都看得更清楚
  • 比解释复杂的远程桌面软件更容易

三步开始使用。

第一次使用时，按这个顺序做最省心。

1

下载并运行 DeskGo

在需要被访问的电脑上下载对应平台的客户端，先手动启动一次，确认能成功连接并看到会话名。

2

记住会话名

启动后会显示会话名和访问链接。常用电脑可以再用一键脚本把会话名固定下来，方便以后重复使用。

3

在浏览器输入会话名

打开浏览器输入会话名即可进入桌面；如果你分享的是完整链接，对方甚至不需要手动输入。
