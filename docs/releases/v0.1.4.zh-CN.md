# DeskGo v0.1.4 发布说明

## 发布定位

`v0.1.4` 是一次聚焦首页体验的小版本修复发布。它把 macOS Desktop CLI 的独立下载区重新放回首页顶部附近，避免 Mac 用户还要滚到通用下载区域里再找对应版本。

## 亮点

### 1. 首页顶部重新出现 macOS 下载区

- 中文和英文首页现在都在主视觉区域附近恢复了独立的 macOS 快速下载块
- Intel（`AMD64`）和 Apple Silicon（`ARM64`）版本都会直接展示出来
- 这个快速下载块仍然保留跳转到完整下载区的入口，方便继续查看全部平台版本

### 2. 最终用户下载路径重新变得直接

- Mac 用户现在进入首页后第一屏附近就能拿到 Desktop CLI，不需要再去通用下载区里翻找
- 这个快速入口继续保持 `v0.1.3` 那套面向最终用户的首页结构，而不是回到技术导向布局

## 验证

发布前已完成以下验证：

- `go build ./cmd/relay`
- 本地渲染 `/`，确认页面包含 `macOS CLI 下载`、`下载 AMD64`、`下载 ARM64`
- 本地渲染 `/en`，确认页面包含 `macOS CLI download`、`Download AMD64`、`Download ARM64`

## 已知限制

- 这个版本只恢复了首页顶部的 macOS 下载入口，不涉及底层桌面端或 Relay 二进制逻辑变更
- Relay 依然更适合部署在反向代理、隧道或其它访问控制层之后
